# main.js
## canvas api 부연설명

### drawChart() 함수
+ 캔버스를 초기화하고 실제 막대 그래프를 그리는 메인 함수
+ 각 데이터에 대해 녹색(#4CAF50) 막대를 그림
+ 막대 위에 실제 값을 표시하고, 아래에 인덱스 번호를 표시
+ 막대의 높이는 최대값을 기준으로 상대적 비율로 계산

### drawAxes() 함수
+ 차트의 X축과 Y축을 그리는 함수
+ 검은색 실선으로 두 축을 그림
+ 축의 굵기는 2픽셀로 설정
+ X축과 Y축이 만나는 지점에서 약간 확장된 형태로 그려짐

### drawGridLines() 함수
+ Y축을 따라 회색(#e0e0e0) 격자선을 그리는 함수
+ 총 5개의 등간격 눈금선을 그림
+ 각 눈금선 왼쪽에 해당하는 값을 표시
+ 데이터의 최대값을 기준으로 눈금 간격이 자동으로 계산
